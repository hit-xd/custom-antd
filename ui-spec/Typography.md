# Typography 文字

**中文名称：** 文字
**英文名称：** Typography
**所属分类：** 全局样式
**一句话简介：** 统一字体、字号层级、行高、字重和文字颜色。

---

## 字体 Font Family

```css
font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", "PingFang SC",
  "Hiragino Sans GB", "Microsoft YaHei", "Helvetica Neue", Helvetica, Arial,
  sans-serif;
```

兜底：`Microsoft YaHei`

---

## 字号与行高

| 类型 | 字号 | 行高 | 字重 | 典型用途 |
|------|------|------|------|----------|
| 重要数字 | `32px` | `40px` | `500` | 重点统计数据 |
| 强调数字 | `28px` | `36px` | `500` | 卡片重点数字 |
| 核心数字 | `24px` | `32px` | `500` | 页面核心数字 |
| 重点文字 | `20px` | `28px` | `500` | 页面标题 |
| 大标题 | `18px` | `26px` | `500` | 模块标题 |
| 模块标题 | `16px` | `24px` | `500` | 分组/卡片标题 |
| 资产配置正文 | `15px` | `28px` | `500` | 资产配置业务正文 |
| 正文 | `14px` | `22px` | `400` | 常规正文、按钮、表单 |
| 辅助文字 | `12px` | `20px` | `400` | 辅助说明、标签 |

---

## 图标尺寸

| 类型 | 尺寸 | 典型用途 |
|------|------|----------|
| 小图标 | `12px` | 输入框、表格、辅助说明 |
| 常规图标 | `16px` | 按钮、导航、列表、表单 |
| 强调图标 | `20px` | 重要状态、模块入口 |
| 大图标 | `24px` | 空状态、结果页、引导页 |

## 文字颜色

### 层级色（浅色背景）

| 类型 | CSS 值 | 典型用途 |
|------|--------|----------|
| 标题/主要 | `color: rgba(0,0,0,0.85)` | 标题、正文重点、字段值 |
| 次要 | `color: rgba(0,0,0,0.65)` | 说明、次级信息 |
| 占位 | `color: rgba(0,0,0,0.45)` | placeholder |
| 禁用 | `color: rgba(0,0,0,0.25)` | 禁用态文字 |

### 层级色（深色背景）

| 类型 | CSS 值 |
|------|--------|
| 主要 | `color: rgba(255,255,255,1)` |
| 次要 | `color: rgba(255,255,255,0.65)` |
| 占位 | `color: rgba(255,255,255,0.45)` |
| 禁用 | `color: rgba(255,255,255,0.25)` |

### 状态色

| 类型 | CSS 值 | 典型用途 |
|------|--------|----------|
| 链接 | `color: #1874FF` | 可点击链接、查看详情 |
| 成功 | `color: #00C078` | 成功、通过 |
| 警告 | `color: #FFA000` | 风险提醒、待确认 |
| 错误 | `color: #FF4D4F` | 失败、校验错误 |

### 次要链接

| 状态 | CSS 值 |
|------|--------|
| 常态 | `color: rgba(0,0,0,0.85)`（深色背景用白色） |
| hover | `color: #1874FF` |

---

## 注意事项

1. 同一页面字号层级控制在 3 个以内。
2. 深色背景使用白色透明度体系，不沿用浅色背景的黑色透明度。
3. 状态色语义保持一致，成功/警告/错误不混用。
