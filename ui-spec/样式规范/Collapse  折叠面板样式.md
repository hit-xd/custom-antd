# 折叠面板 Collapse — 样式规范

## 面板容器

| 样式属性 | 值 |
|---|---|
| 背景色 | `#FFFFFF` |
| 边框颜色 | `#D9D9D9` |
| 边框宽度 | `1px` |
| 圆角 | `2px` |
| 项间分割线颜色 | `#E5E6EB` |
| 项间分割线高度 | `1px` |

## 标题栏

| 样式属性 | 值 |
|---|---|
| 高度 | `48px` |
| 背景色 | `#F5F7FA` |
| 文字颜色 | `#000000` 85% |
| 字号 | `14px` |
| 字重 | Regular |
| 行高 | `22px` |
| 文字垂直间距 | `13px` |
| 文字水平间距 | `16px` |

## 展开图标

| 样式属性 | 值 |
|---|---|
| 尺寸 | `12px × 12px` |
| 颜色 | `#000000` 45% |
| 与标题间距 | `12px` |

### 位置

- **左收起**：图标在文字左侧，距左边框 `16px`
- **右收起**：图标在右侧，距右边框 `16px`

### 禁用状态图标颜色

`#000000` 25%

## 交互状态

| 状态 | 样式属性 | 值 |
|---|---|---|
| **默认** | 标题栏背景色 | `#F5F7FA` |
| | 标题文字颜色 | `#000000` 85% |

| **禁用** | 标题文字颜色 | `#000000` 25% |
