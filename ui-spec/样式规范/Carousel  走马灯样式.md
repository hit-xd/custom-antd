# 走马灯 Carousel — 样式规范

## 轮播容器

| 样式属性 | 值 |
|---|---|
| 占位背景色 | `#D9D9D9` |
| 容器圆角 | `0px ~ 4px` |
| 常用比例 | `16:9`、`4:3`、`2:1` |
| 超出隐藏 | 不出现横向滚动条 |

## 指示器

| 样式属性 | 值 |
|---|---|
| 距容器底部距离 | `16px` |
| 指示器水平间距 | `6px ~ 8px` |

### 默认圆点

| 样式属性 | 值 |
|---|---|
| 尺寸 | `6px × 6px` |
| 颜色 | `rgba(255, 255, 255, 0.45)` |
| 圆角 | `50%` |

### 当前指示器（长条样式）

| 样式属性 | 值 |
|---|---|
| 尺寸 | `24px × 4px` |
| 圆角 | `2px` |
| 颜色 | `#FFFFFF` |

> 若使用圆点样式：尺寸保持 `6px × 6px`，颜色改为 `#FFFFFF`

## 切换箭头

| 样式属性 | 值 |
|---|---|
| 箭头按钮尺寸 | `32px × 32px` |
| 箭头图标尺寸 | `16px × 16px` |
| 图标颜色 | `#FFFFFF` |
| 按钮背景色 | `rgba(0, 0, 0, 0.25)` |
| 按钮圆角 | `50%` |
| 左箭头距左侧 | `16px` |
| 右箭头距右侧 | `16px` |
| 垂直居中 | – |
