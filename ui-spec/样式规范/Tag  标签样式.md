# 标签 Tag — 样式规范

## 通用

- 标签之间水平间距：`8px`

## 基础标签

### 无边框普通

| 样式属性 | 值 |
|---|---|
| 高度 | `20px`，宽度自适应 |
| 背景色 | `#F4F6F8` |
| 边框 | 无 |
| 圆角 | `2px` |
| 文字颜色 | `#000000` 65% |
| 字号 | `14px` |
| 字重 | Regular |
| 行高 | `22px` |
| 文字垂直间距 | `1px` |
| 文字水平间距 | `8px` |

### 有边框普通

| 样式属性 | 值 |
|---|---|
| 高度 | `20px`，宽度自适应 |
| 背景色 | `#FFFFFF` |
| 边框颜色 | `#000000` 25% |
| 边框宽度 | `1px` |
| 圆角 | `2px` |
| 文字颜色 | `#000000` 65% |
| 字号 | `14px` |
| 字重 | Regular |
| 行高 | `22px` |
| 文字垂直间距 | `1px` |
| 文字水平间距 | `8px` |

### 带图标

- 图标尺寸：`14px × 14px`，颜色跟随文字颜色
- 图标与文字间距：`4px`
- 整体水平间距：`8px`

### 带关闭图标

- 关闭图标尺寸：`12px × 12px`，颜色：`#000000` 45%
- 关闭图标与文字间距：`4px`
- 整体水平间距：`8px`

## 一级标签（实底斜切）

| 样式属性 | 值 |
|---|---|
| 高度 | `20px`，宽度自适应 |
| 边框 | 无 |
| 背景色 | 使用对应语义色 |
| 斜切 | `clip-path: polygon(4px 0, 100% 0, calc(100% - 4px) 100%, 0 100%)` |
| 文字颜色 | `#FFFFFF` |
| 字号 | `14px` |
| 字重 | Regular |
| 行高 | `22px` |
| 文字垂直间距 | `1px` |
| 文字水平间距 | `12px` |
| 关闭图标尺寸 | `12px × 12px` |
| 关闭图标颜色 | `#FFFFFF` |
| 关闭图标与文字间距 | `4px` |

### 常用颜色

| 类型 | 背景色 | 文字色 |
|---|---|---|
| 红色 | `#FF4D4F` | `#FFFFFF` |
| 金色 | `#9B7235` | `#FFFFFF` |
| 绿色 | `#009966` | `#FFFFFF` |
| 蓝色 | `#1874FF` | `#FFFFFF` |
