# 气泡卡片 Popover — 样式规范

## 卡片容器

| 样式属性 | 值 |
|---|---|
| 最大宽度 | `700px` |
| 背景色 | `#FFFFFF` |
| 圆角 | `2px` |
| 边框颜色 | `#F0F0F0` 或无边框 |
| 阴影 | `0 3px 7px 0 rgba(0,0,0,0.12), 0 6px 16px 0 rgba(0,0,0,0.08), 0 9px 28px 0 rgba(0,0,0,0.05)` |
| 与触发元素间距 | `4px ~ 8px` |

## 标题

| 样式属性 | 值 |
|---|---|
| 最小高度 | `40px`，高度自适应 |
| 文字颜色 | `#000000` 85% |
| 字号 | `16px` |
| 字重 | Regular |
| 行高 | `24px` |
| 上下间距 | `8px` |
| 左右间距 | `16px` |
| 下方分割线颜色 | `#000000` 15%（`#F0F0F0`） |
| 分割线高度 | `1px` |

## 内容

| 样式属性 | 值 |
|---|---|
| 高度 | 自适应 |
| 文字颜色 | `#000000` 65% |
| 字号 | `14px` |
| 字重 | Regular |
| 行高 | `22px` |
| 上下间距 | `12px` |
| 左右间距 | `16px` |
| 超宽处理 | 自动换行，左对齐 |

## 箭头

| 样式属性 | 值 |
|---|---|
| 箭头尺寸 | `10px × 6px` |
| 箭头颜色 | `#FFFFFF` |
| 阴影 | 与卡片一致 |
