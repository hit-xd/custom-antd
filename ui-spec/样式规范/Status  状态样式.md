# 状态 Status — 样式规范

## 状态圆点

| 样式属性 | 值 |
|---|---|
| 圆点尺寸 | `6px × 6px` |
| 形状 | 正圆形 |
| 圆点与文字间距 | `6px` |
| 状态项水平间距 | `24px` |

### 常用状态颜色

| 状态 | 圆点颜色 |
|---|---|
| 已完成 | `#00C078` |
| 处理中 | `#1874FF` |
| 警示 | `#FFA000` |
| 默认 | `#BFBFBF` |
| 失败 | `#FF4D4F` |

## 状态文字

| 样式属性 | 值 |
|---|---|
| 文字颜色 | `#000000` 65% |
| 字号 | `14px` |
| 字重 | Regular |
| 行高 | `22px` |
| 字间距 | `0px` |
