# 文字提示 Tooltip — 样式规范

## 提示气泡

| 样式属性 | 值 |
|---|---|
| 最大宽度 | `350px` |
| 最小高度 | `32px` |
| 背景色 | `rgba(0, 0, 0, 0.80)` |
| 圆角 | `2px` |
| 阴影 | `0 3px 7px 0 rgba(0,0,0,0.12), 0 6px 16px 0 rgba(0,0,0,0.08), 0 9px 28px 0 rgba(0,0,0,0.05)` |
| 与触发元素间距 | `4px ~ 8px` |
| 距页面边缘安全距离 | `≥ 8px` |
