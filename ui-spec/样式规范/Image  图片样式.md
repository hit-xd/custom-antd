# 图片 Image — 样式规范

## 图片预览失败

| 样式属性 | 值 |
|---|---|
| 占位区域背景色 | `#F5F5F5` |
| 占位图标尺寸 | `40px × 40px` |
| 图标颜色 | `#D9D9D9` |
| 提示文字颜色 | `#000000` 25% |
| 字号 | `14px` |
| 字重 | Regular |
| 行高 | `22px` |
| 图标与文字间距 | `8px` |

## 图片比例

- 常用比例：`1:1`、`3:2`、`4:3`、`16:9`、`2:1`
- 图片容器背景色：`#F5F5F5`
- 图片圆角：`2px ~ 4px`

| 工具栏高度 | `48px` |
| 工具栏背景色 | `rgba(0, 0, 0, 0.45)` |
| 操作图标尺寸 | `16px × 16px` |
| 图标颜色 | `#FFFFFF` |
| 图标水平间距 | `24px` |
| 关闭图标距右侧边缘 | `24px` |

| 状态 | 样式属性 | 值 |
