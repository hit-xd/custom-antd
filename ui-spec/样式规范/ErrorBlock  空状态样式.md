# 空状态 ErrorBlock — 样式规范

## 默认空状态

| 样式属性 | 值 |
|---|---|
| 占位插图尺寸 | `64px × 64px` |
| 提示文字颜色 | `#000000` 25% |
| 字号 | `14px` |
| 字重 | Regular |
| 行高 | `22px` |
| 字间距 | `0px` |
| 插图与文字垂直间距 | `8px` |
| 对齐方式 | 水平居中、垂直居中 |

## 表格空状态

| 样式属性 | 值 |
|---|---|
| 空状态区域高度 | `≥ 160px` |
| 占位插图尺寸 | `48px × 48px` |
| 提示文字颜色 | `#000000` 25% |
| 字号 | `14px` |
| 行高 | `22px` |

## 日历空状态

| 样式属性 | 值 |
|---|---|
| 图标尺寸 | `32px × 32px` |
| 提示文字颜色 | `#000000` 25% |
| 字号 | `12px` |
| 行高 | `20px` |

## 业务型空状态

| 样式属性 | 值 |
|---|---|
| 占位插图尺寸 | `160px × 120px` 或自适应 |
| 提示标题颜色 | `#000000` 85% |
| 标题字号 | `14px`，字重：Medium，行高：`22px` |
| 说明文字颜色 | `#000000` 45% |
| 说明字号 | `12px`，字重：Regular，行高：`20px` |
| 插图与标题垂直间距 | `16px` |
| 标题与说明文字垂直间距 | `4px` |

## 带操作按钮空状态

- 按钮高度：`32px`，宽度自适应
- 按钮背景色：`#9B7235`，文字颜色：`#FFFFFF`，圆角：`2px`
- 按钮文字字号：`14px`，字重：Regular，行高：`22px`
- 按钮与说明文字垂直间距：`16px`
