# 徽标数 Badge — 样式规范

## 文字徽标

| 样式属性 | 值 |
|---|---|
| 高度 | `16px`，宽度自适应 |
| 背景色 | `#FF4D4F` |
| 圆角 | `8px` |
| 边框 | 无 |
| 文字颜色 | `#FFFFFF` |
| 字号 | `12px` |
| 字重 | Regular |
| 行高 | `16px` |
| 文字垂直间距 | `0px` |
| 文字水平间距 | `6px` |

## 数字徽标

| 样式属性 | 值 |
|---|---|
| 最小宽度 | `16px` |
| 高度 | `16px` |
| 个位数时 | `16px × 16px`（正圆形） |
| 多位数时 | 宽度自适应 |
| 背景色 | `#FF4D4F` |
| 圆角 | `8px` |
| 边框 | 无 |
| 数字颜色 | `#FFFFFF` |
| 字号 | `12px` |
| 字重 | Regular |
| 行高 | `16px` |
| 数字垂直间距 | `0px` |
| 数字水平间距 | `8px` |

## 小红点徽标

| 样式属性 | 值 |
|---|---|
| 尺寸 | `6px × 6px` |
| 形状 | 正圆形 |
| 背景色 | `#FF4D4F` |
| 文字 | 无 |
| 边框 | 无 |
