# 头像 Avatar — 样式规范

## 尺寸规范

| 类型 | 尺寸 |
|---|---|
| 小号 | `24px × 24px` |
| 默认 | `32px × 32px` |
| 大号 | `48px × 48px` |
| 超大 | `64px × 64px` |

图标尺寸建议为头像尺寸的 `50%`。

## 圆形头像

| 样式属性 | 值 |
|---|---|
| 尺寸 | `32px × 32px` |
| 圆角 | `50%` |
| 背景色 | `#F5F5F5` |
| 图片裁切 | 居中裁切，铺满 |

### 图标头像

- 背景色：`#E6F4FF` / `#FFF1F0` / `#F5F5F5`
- 图标尺寸：`20px × 20px`
- 图标颜色：对应主题色或 `#1677FF`
- 居中对齐

### 字符头像

- 背景色：`#1677FF` 或 `#F5F5F5`
- 文字颜色：`#FFFFFF` 或 `#000000` 65%
- 字号：`16px`，字重：Regular，行高：`24px`
- 水平垂直居中

## 方形头像

| 样式属性 | 值 |
|---|---|
| 尺寸 | `32px × 32px` |
| 圆角 | `4px` |
| 背景色 | `#F5F5F5` |

### 图标方形

- 图标尺寸：`16px × 16px`
- 水平垂直居中

### 字符方形

- 字号：`16px`，字重：Regular，行高：`24px`
- 水平垂直居中

## 组合头像

| 样式属性 | 值 |
|---|---|
| 单个头像尺寸 | `32px × 32px` |
| 重叠距离 | `8px ~ 12px` |
| 头像边框颜色 | `#FFFFFF` |
| 边框宽度 | `1px` |
| 圆形圆角 | `50%` |
| 方形圆角 | `4px` |

### "+N" 更多头像

- 尺寸：`32px × 32px`
- 背景色：`#F5F5F5`
- 文字颜色：`#000000` 65%
- 字号：`16px`，行高：`24px`

## 交互状态

| 状态 | 样式属性 | 值 |
|---|---|---|
| **加载失败** | 默认背景色 | `#F5F5F5` |
| | 默认图标颜色 | `#000000` 25% |
| **hover（可点击）** | 遮罩颜色 | `rgba(0, 0, 0, 0.25)` |
| | 鼠标指针 | `pointer` |
