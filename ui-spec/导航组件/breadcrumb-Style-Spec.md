# 面包屑 / Breadcrumb 样式规范
# 1. 结构组成

面包屑组件由以下部分组成：

1. 外层面包屑容器
2. 普通层级项
3. 当前页面项
4. 分隔符

## 1.1 外层面包屑容器

外层容器用于承载所有层级项和分隔符。

整体高度：`22px`  
整体宽度：自适应  
背景：无填充  
描边：无  
布局方向：横向排列  
垂直对齐：居中对齐

## 1.2 普通层级项

普通层级项用于表示当前页面之前的上级页面。

普通层级项一般可点击，用于返回对应层级页面。

普通层级项文字内容示例：`首页`

## 1.3 当前页面项

当前页面项用于表示用户当前所在页面。

当前页面项通常不可点击，仅用于标识当前位置。

当前页面项文字内容示例：`当前页面`

## 1.4 分隔符

分隔符用于分隔不同层级项。

分隔符样式：`/`  
分隔符颜色：`#000000 45%`
分隔符与前后元素的间距：`8px`

---

# 2. 状态规则

## 2.1 普通页面状态

普通页面状态用于当前页面之前的层级项。

字体颜色：`#000000 45%` 
字体字号：`14px`  
字体行高：`22px`  
字间距：`0px`  
字重：`Regular`


hover 状态
hover 状态文字颜色：`#000000 85%`  
字体字号：`14px`  
字体行高：`22px`  
字间距：`0px`  
字重：`Regular`

## 2.2 当前页面状态

当前页面状态用于最后一级路径，表示当前所在页面。

字体颜色：`#000000 85%` 
字体字号：`14px`  
字体行高：`22px`  
字间距：`0px`  
字重：`Regular`

当前页面项通常不可点击。


## 2.3 分隔符状态

分隔符用于区分不同层级。

分隔符内容：`/`  
分隔符颜色：`#000000 45%` / `rgba(0, 0, 0, 0.45)`  
分隔符字号：跟随面包屑文字字号，`14px`  
分隔符行高：跟随面包屑整体行高，`22px`  
分隔符左右间距：`8px`
---

# 3. 尺寸 / 间距 / 颜色 / 交互规则

## 3.1 尺寸规则

整体高度：`22px`  
整体宽度：自适应  
单个文本项高度：`22px`  
文字字号：`14px`  
文字行高：`22px`  
字间距：`0px`

## 3.2 间距规则

元素间距：统一为 `8px`  
普通层级项与分隔符之间的间距：`8px`  
分隔符与后一个层级项之间的间距：`8px`  
组件上下内边距：`0px`，或由所在页面布局控制  
组件左右内边距：`0px`，或由所在页面布局控制

## 3.3 颜色规则

普通层级项文字颜色：`#000000 45%`
当前页面项文字颜色：`#000000 85%`
分隔符颜色：`#000000 45%`
背景色：透明 / 无填充  
描边颜色：无

## 3.4 交互规则

普通层级项点击行为：跳转至对应层级页面  
当前页面项点击行为：不可点击  
分隔符点击行为：不可点击
