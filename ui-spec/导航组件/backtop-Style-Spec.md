# 返回顶部 / BackTop样式规范

## 结构组成

返回顶部组件由以下部分组成：

1. 外层固定定位容器；
2. 方形按钮容器；
3. 向上箭头图标；
4. hover 状态反馈。

结构关系：

- 按钮整体固定在页面或指定滚动容器右下角；
- 图标位于按钮中心；
- 图标水平居中、垂直居中；
- 点击按钮后，页面或指定滚动容器平滑滚动至顶部。

---

## 状态规则

### 默认状态

默认状态用于组件显示后、用户未操作时的视觉表现。

- 按钮可点击；
- 图标居中显示；
- 按钮背景为黑色半透明；
- 按钮无明显描边；
- 按钮固定在页面或滚动容器右下角。

### 悬停状态

悬停状态用于鼠标移入按钮时的视觉反馈。

- 按钮背景透明度建议加深；
- 图标颜色保持白色；
- 鼠标样式为 `pointer`；
- 可增加轻微阴影反馈。

### 隐藏状态

页面或滚动容器未达到滚动阈值时，组件隐藏。

- 不占据页面布局空间；
- 不影响页面原有排版；
- 隐藏后不可点击；
- 建议使用透明度过渡完成显示 / 隐藏。

## 尺寸 / 间距 / 颜色 / 交互规则

### 整体定位

组件定位方式：`fixed` 或相对于指定滚动容器定位  
默认显示位置：右下角  
距离页面右侧边距：`24px`  
距离页面底部边距：`80px`


### 滚动阈值

组件默认滚动阈值：`1080px`

显示规则：

- 当页面或目标滚动容器垂直滚动距离大于等于 `1080px` 时显示；
- 当滚动距离小于 `1080px` 时隐藏；

### 按钮尺寸

按钮宽度：`40px`  
按钮高度：`40px`  
按钮圆角：`2px`  

### 按钮内部图标尺寸
图表引用图标资源库中的返回顶部图标
图标尺寸：`18px × 20px`  
图标与外部按钮水平居中对齐

### 按钮默认样式

按钮背景色：`#000000 30%`  
按钮边框：无明显边框  
按钮阴影：无明显阴影  
按钮圆角：`2px`

图标颜色：`#FFFFFF`  

- 图标左侧边距 `11px`
- 图标右侧边距 `11px`
- 图标顶部边距 `10px`
- 图标底部边距 `10px`

### 按钮悬停样式

hover 背景色：`#000000 45%`  
hover 图标颜色：`#FFFFFF`  
hover 边框：无  
hover 阴影：`0 2px 8px  #000000 15%`

### 按钮按下样式

active 背景色：`#000000 55%`  
active 图标颜色：`#FFFFFF`  
active 边框：无  
active 阴影：无明显增强

### 滚动交互

点击按钮后滚动目标：

- 默认滚动页面 `window`；
- 若指定容器，则滚动对应容器；
- 滚动目标位置为顶部 `0px`。

滚动动画：

- 建议使用平滑滚动；
- 动画时长建议为 `300ms - 450ms`；
- 默认建议使用 `450ms`；
- 滚动过程中不需要额外 loading 状态；
- 滚动到顶部后，组件根据阈值自动隐藏。
