# 标签页 Tabs 样式规范

## 2. 一级标签页

### 2.3 结构组成

一级标签页由以下部分组成：

- 标签容器。
- 标签项。
- 标签文字。
- 可选前置图标。
- 可选后置说明 icon。
- 底部分割线。
- 当前选中项底部指示线。

### 2.4 状态规则

一级标签页包含以下状态：

- 默认态：未选中、可点击。
- 选中态：当前页面所在的一级分类。
- 禁用态：不可点击。
- hover 态：可点击，鼠标悬停标签页之上

带说明 icon 的标签项：

- 说明 icon 放在标签文字后方。
- 鼠标 hover 到说明 icon 后出现气泡卡片。
- 气泡卡片样式：文字提示 / Tooltip 

### 2.5 尺寸 / 间距 / 颜色 / 交互规则

#### 整体样式

- 整体高度：`48px`
- 宽度：自适应
- 背景：无填充
- 底部分割线高度：`1px`
- 底部分割线颜色：`#000000 15%`
- 标签排列方向：横向
- 标签之间的水平间距：`24px`

#### 单个标签样式

- 标签高度：`48px`
- 标签文字字号：`16px`
- 标签文字字间距：`0px`
- 标签文字行高：`24px`
- 标签宽度：根据文字内容自适应
- 标签左右内边距：`0px`
- 标签上下内边距：`12px`
- 标签文字与前置图标的水平间距：`4px`
- 标签文字与后置说明 icon 的水平间距：`4px`

#### 选中态

- 文字颜色：`#9B7235`
- 文字字重：`Bold` 
- 底部指示线高度：`2px`
- 底部指示线颜色：`#9B7235`
- 前置图标颜色：`#9B7235`
- 后置说明 icon 颜色：`#9B7235`

#### 默认态

- 文字颜色：`#000000 85%`
- 前置图标颜色：`#000000 65%`
- 后置说明 icon 颜色：`#000000 65%`
- 背景：无填充
- 底部指示线：无

#### 禁用态

- 文字颜色：`#000000 25%`
- 图标颜色：`#000000 25%`
- 不显示底部指示线
- 不响应点击

#### 交互规则

- hover 状态文字颜色：`#9B7235`
- hover 状态图标颜色：`#9B7235`
- 点击后切换当前选中项
- 标签较多时是否支持横向滚动：支持
