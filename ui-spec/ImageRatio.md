# ImageRatio 图片比例

**中文名称：** 图片比例
**英文名称：** ImageRatio
**所属分类：** 全局样式
**一句话简介：** 统一图片容器比例标准，定义 1:1、4:3、3:2、16:9、2:1 五类核心比例。

---

## 1:1 正方形

CSS 参数：`aspect-ratio: 1 / 1`（兼容：`padding-bottom: 100%`）
- 适用场景：头像、图标、宫格入口、产品图标、小型封面缩略图
- 使用约束：图片主体居中；头像可配合 `border-radius: 50%`

---

## 4:3 经典矩形

CSS 参数：`aspect-ratio: 4 / 3`（兼容：`padding-bottom: 75%`）
- 适用场景：内容卡片封面、业务说明插图、产品介绍图、列表卡片封面、证件预览
- 使用约束：同一组卡片图片区域高度须一致

---

## 3:2 黄金矩形

CSS 参数：`aspect-ratio: 3 / 2`（兼容：`padding-bottom: 66.6667%`）
- 适用场景：客户故事、活动图、产品封面、营销配图、中型 Banner、图文列表缩略图
- 使用约束：同一级卡片内不与 4:3 混排

---

## 16:9 宽屏矩形

CSS 参数：`aspect-ratio: 16 / 9`（兼容：`padding-bottom: 56.25%`）
- 适用场景：视频封面、页面 Banner、首页头图、运营横幅、数据看板配图
- 使用约束：主体不宜贴左右边缘；叠加文字时须加遮罩或渐变保证可读性

---

## 2:1 超宽矩形

CSS 参数：`aspect-ratio: 2 / 1`（兼容：`padding-bottom: 50%`）
- 适用场景：窄 Banner、运营横幅、列表顶部宣传图、模块引导图、跨栏宣传图
- 使用约束：不适合复杂内容详情；核心元素居中，文字数量须控制

---

## CSS 实现

优先使用 `aspect-ratio`；兼容方案容器设 `position: relative`，子内容设 `position: absolute`。

```css
.ratio-1-1   { aspect-ratio: 1 / 1; }
.ratio-4-3   { aspect-ratio: 4 / 3; }
.ratio-3-2   { aspect-ratio: 3 / 2; }
.ratio-16-9  { aspect-ratio: 16 / 9; }
.ratio-2-1   { aspect-ratio: 2 / 1; }

/* 兼容方案 */
.ratio-1-1   { padding-bottom: 100%; }
.ratio-4-3   { padding-bottom: 75%; }
.ratio-3-2   { padding-bottom: 66.6667%; }
.ratio-16-9  { padding-bottom: 56.25%; }
.ratio-2-1   { padding-bottom: 50%; }
```

---

## 场景速查

| 比例 | aspect-ratio | padding-bottom | 典型场景 |
|------|--------------|----------------|----------|
| 1:1 | `1 / 1` | `100%` | 头像、图标、宫格入口 |
| 4:3 | `4 / 3` | `75%` | 卡片封面、产品介绍、证件预览 |
| 3:2 | `3 / 2` | `66.6667%` | 活动图、营销配图、中型 Banner |
| 16:9 | `16 / 9` | `56.25%` | 视频封面、首页头图、运营横幅 |
| 2:1 | `2 / 1` | `50%` | 窄 Banner、列表顶部宣传图 |

---

## 注意事项

1. 比例由容器控制，不依赖图片原始尺寸撑开布局。
2. 同一页面/同级列表内保持比例统一，避免混用。
3. 兼容 `padding-bottom` 方案时，子内容须绝对定位，避免撑开容器。
