# 多选框 / Checkbox - 前端样式规范

> 面向对象：前端开发。本文档仅包含组件的 UI 样式规则和状态样式，用于指导 antd 组件样式调整以达到设计规范要求。

## 1. 组件结构概览

Checkbox 支持**基础样式**和**卡片式样式**两种形态。

**基础 Checkbox** 包含：选择框（方形）、选项文字、后置说明 icon（可选）、Tooltip（可选）。

**卡片式 Checkbox** 包含：卡片容器、前置图标（可选）、选项文字、后置说明 icon（可选）、辅助说明文字（可选）、选中角标（可选，位于右下角）。卡片式不展示方形选择框，选中态通过边框、文字颜色或角标表达。卡片式不展示半选状态。

## 2. 尺寸与间距规范

### 2.1 基础 Checkbox

| 项目 | 数值 |
|---|---:|
| 选项内容高度 | `24px` |
| 建议点击热区高度 | `≥32px` |
| 选择框尺寸 | `16px × 16px` |
| 选择框圆角 | `2px` |
| 勾选线宽 | `2px` |
| 半选横线尺寸 | `8px × 2px` |
| 选择框与文字间距 | `8px` |
| 文字字号 | `14px` |
| 文字行高 | `24px` |
| 横向选项间距 | `24px` |
| 纵向选项间距 | `8px` |
| 说明 icon 尺寸 | `16px × 16px` |
| 文字与说明 icon 间距 | `8px` |

### 2.2 卡片式 Checkbox

| 项目 | 数值 |
|---|---:|
| 基础卡片高度（单行） | `32px` |
| 卡片圆角 | `2px` |
| 卡片边框 | `1px` |
| 左右内边距 | `16px` |
| 上下内边距 | `4px` |
| 文字字号 | `14px` |
| 文字行高 | `24px` |
| 前置图标尺寸 | `16px × 16px` |
| 图标与文字间距 | `8px` |
| 文字与后置说明 icon 间距 | `8px` |
| 卡片横向间距 | `8px` |
| 卡片纵向间距 | `8px` |

### 2.3 带辅助说明的卡片高度

| 场景 | 建议高度 |
|---|---:|
| 单行主文案 | `32px` |
| 主文案 + 一行说明 | `64px` |
| 主文案 + 两行说明 | `80px` 起 |

### 2.4 辅助说明文字

| 项目 | 数值 |
|---|---:|
| 字号 | `12px` |
| 行高 | `20px` |
| 颜色 | 45% `#000000` |
| 与主文案垂直间距 | `2px` |

## 3. 颜色规范

| 状态 | 选择框/卡片背景 | 边框色 | 文字色 | 图标色 |
|---|---|---|---|---|
| 未选中 | `#FFFFFF` | 25% `#000000` | 85% `#000000` | 45% `#000000` |
| Hover | `#FFFFFF` | `#9B7235` | 85% `#000000` | `#9B7235` |
| 选中（基础） | `#9B7235` | `#9B7235` | 85% `#000000` | 勾选 `#FFFFFF` |
| 选中（卡片） | `#FFFFFF` | `#9B7235` | `#9B7235` | `#9B7235` |
| 半选 | `#9B7235` | `#9B7235` | 85% `#000000` | 横线 `#FFFFFF` |
| 禁用未选中 | `#F4F6F8` | 25% `#000000` | 25% `#000000` | 25% `#000000` |
| 禁用选中 | `#F4F6F8` | 25% `#000000` | 25% `#000000` | 25% `#000000` |

说明 icon：默认 45% `#000000`，Hover `#9B7235`，Tooltip 最大宽度 `280px`

前置图标：默认 45% `#000000`，选中/卡片选中 `#9B7235`，禁用 25% `#000000`

## 5. 状态样式规范

### 5.1 基础 Checkbox 未选中
- 选择框背景 `#FFFFFF` | 边框 25% `#000000` | 文字 85% `#000000` | 说明 icon 45% `#000000`

### 5.2 基础 Checkbox Hover
- 鼠标 `pointer` | 选择框边框 `#9B7235` | 选择框背景 `#FFFFFF` | 文字 85% `#000000` | 说明 icon 可为 `#9B7235`
- Hover 作用于整个选项点击区域

### 5.3 基础 Checkbox 选中
- 选择框背景 `#9B7235` | 边框 `#9B7235` | 勾选标记 `#FFFFFF`（线宽 `2px`） | 文字 85% `#000000`
- 选中后不改变文字颜色，状态识别主要依赖选择框

### 5.4 半选
- 选择框背景 `#9B7235` | 边框 `#9B7235` | 白色横线 `8px × 2px` | 文字 85% `#000000`
- 用于全选/父子联动场景，由数据计算得出

### 5.5 基础 Checkbox 禁用未选中
- 选择框背景 `#F4F6F8` | 边框 25% `#000000` | 文字/icon 25% `#000000` | 鼠标 `not-allowed` | 不展示 Hover

### 5.6 基础 Checkbox 禁用选中
- 选择框背景 `#F4F6F8` | 边框 25% `#000000` | 勾选标记/文字 25% `#000000` | 鼠标 `not-allowed`

### 5.7 卡片 Checkbox 未选中
- 背景 `#FFFFFF` | 边框 25% `#000000` | 文字 85% `#000000` | 前置图标 45% `#000000`

### 5.8 卡片 Checkbox Hover
- 鼠标 `pointer` | 边框 `#9B7235` | 背景 `#FFFFFF` | 文字 85% `#000000` | 前置/后置 icon 可为 `#9B7235`

### 5.9 卡片 Checkbox 选中
- 背景 `#FFFFFF` | 边框 `#9B7235` | 文字 `#9B7235` | 前置图标 `#9B7235`
- 右下角可展示选中角标，颜色 `#9B7235`
- 主要通过边框与文字颜色共同表达；密度高时可只保留边框与角标

### 5.10 卡片无半选状态
- 卡片式 Checkbox 仅展示未选中、选中、禁用未选中、禁用选中四类状态

### 5.11 卡片 Checkbox 禁用
- 背景 `#F4F6F8` | 边框 25% `#000000` | 文字/图标 25% `#000000` | 鼠标 `not-allowed`

### 5.12 前置图标
- 尺寸 `16px × 16px` | 与文字间距 `8px` | 默认 45% `#000000` | 选中态 `#9B7235` | 禁用 25% `#000000`
- 图标如有品牌色应降低饱和度，避免与选中主色冲突

### 5.13 后置说明 icon 与 Tooltip
- 说明 icon 尺寸 `16px × 16px` | 文字与 icon 间距 `8px` | 默认 45% `#000000` | Hover `#9B7235` | Tooltip 最大宽度 `280px`
- Tooltip 在 Hover 到说明 icon 时展示，非 Hover 整个选项时展示
