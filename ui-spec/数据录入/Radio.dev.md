# 单选框 / Radio - 前端样式规范

> 面向对象：前端开发。本文档仅包含组件的 UI 样式规则和状态样式，用于指导 antd 组件样式调整以达到设计规范要求。

## 1. 组件结构概览

Radio 支持**基础样式**和**卡片式样式**两种形态。

**基础 Radio** 包含：选择符（圆形外圈 + 内部圆点）、选项文字、后置说明 icon（可选）、Tooltip（可选）。

**卡片式 Radio** 包含：卡片容器、前置图标（可选）、选项文字、后置说明 icon（可选）、辅助说明文字（可选）。卡片式不展示圆形选择符，选中态通过边框、文字颜色或浅色背景表达。

## 2. 尺寸与间距规范

| 项目 | 数值 |
|---|---:|

| 建议点击热区高度 | `≥32px` |
| 选择符外圈尺寸 | `16px × 16px` |
| 选中圆点尺寸 | `8px × 8px` |
| 选择符与文字间距 | `8px` |
| 文字字号 | `14px` |
| 文字行高 | `24px` |
| 横向选项间距 | `24px` |
| 纵向选项间距 | `8px` |
| 说明 icon 尺寸 | `16px × 16px` |
| 文字与说明 icon 间距 | `8px` |

### 2.2 卡片式 Radio

| 项目 | 数值 |
|---|---:|
| 基础卡片高度 | `32px` |
| 卡片圆角 | `2px` |
| 卡片边框 | `1px` |
| 左右内边距 | `16px` |
| 上下内边距 | `4px` |
| 文字字号 | `14px` |
| 文字行高 | `24px` |
| 前置图标尺寸 | `16px × 16px` |
| 图标与文字间距 | `8px` |
| 卡片横向间距 | `8px` |
| 卡片纵向间距 | `8px` |
| 带辅助说明卡片内边距 | `12px 16px` |

### 2.3 辅助说明文字

| 项目 | 数值 |
|---|---:|
| 字号 | `12px` |
| 行高 | `20px` |
| 颜色 | 45% `#000000` |

## 3. 颜色规范

| 状态 | 选择符/卡片背景 | 边框色 | 圆点/文字强调色 | 文字色 |
|---|---|---|---|---|
| 未选中 | `#FFFFFF` | 25% `#000000` | — | 85% `#000000` |
| Hover | `#FFFFFF` | `#9B7235` | — | 85% `#000000` |
| 选中（基础） | `#FFFFFF` | `#9B7235` | `#9B7235`（圆点） | 85% `#000000` |
| 选中（卡片） | `#FFFFFF` | `#9B7235` | `#9B7235`（文字） | `#9B7235` |
| 禁用未选中 | `#F4F6F8` | 25% `#000000` | — | 25% `#000000` |
| 禁用选中（基础） | `#F4F6F8` | 25% `#000000` | 25% `#000000` | 25% `#000000` |
| 禁用选中（卡片） | `#F4F6F8` | 45% `#9B7235` | 45% `#9B7235` | 45% `#9B7235` |

图标颜色：
- 说明 icon 默认：45% `#000000`
- 说明 icon Hover：`#9B7235`
- 前置图标默认：45% `#000000`（或业务色）
- 前置图标选中（卡片）：`#9B7235`
- 前置图标禁用：25% `#000000`

| 文本类型 | 字号 | 字重 | 行高 | 颜色 |
|---|---|---|---|---|
| 常规选项文字 | `14px` | Regular | `24px` | 85% `#000000` |
| 卡片选项文字（未选中） | `14px` | Regular | `24px` | 85% `#000000` |
| 卡片选项文字（选中） | `14px` | Regular | `24px` | `#9B7235` |
| 辅助说明文字 | `12px` | Regular | `20px` | 45% `#000000` |
| 禁用文字 | `14px` | Regular | `24px` | 25% `#000000` |

## 5. 状态样式规范

### 5.1 基础 Radio 未选中
- 选择符背景 `#FFFFFF` | 边框 25% `#000000` | 文字 85% `#000000` | 说明 icon 45% `#000000`

### 5.2 基础 Radio Hover
- 鼠标 `pointer` | 选择符边框 `#9B7235` | 选择符背景 `#FFFFFF` | 文字 85% `#000000` | 说明 icon 可为 `#9B7235`
- Hover 作用于整个选项点击区域

### 5.3 基础 Radio 选中
- 选择符背景 `#FFFFFF` | 边框 `#9B7235` | 圆点 `#9B7235`（`8px × 8px`） | 文字 85% `#000000`
- 选中后不建议改变文字颜色，状态识别主要依赖选择符

### 5.4 基础 Radio 禁用未选中
- 选择符背景 `#F4F6F8` | 边框 25% `#000000` | 文字/icon 25% `#000000` | 鼠标 `not-allowed` | 不展示 Hover

### 5.5 基础 Radio 禁用选中
- 选择符背景 `#F4F6F8` | 边框 25% `#000000` | 圆点 25% `#000000` | 文字 25% `#000000` | 鼠标 `not-allowed`

### 5.6 卡片 Radio 未选中
- 背景 `#FFFFFF` | 边框 25% `#000000` | 文字 85% `#000000` | 前置图标 45% `#000000`

### 5.7 卡片 Radio Hover
- 背景 `#FFFFFF` | 边框 `#9B7235` | 文字 85% `#000000` | 鼠标 `pointer`

### 5.8 卡片 Radio 选中
- 背景 `#FFFFFF` | 边框 `#9B7235` | 文字 `#9B7235` | 前置图标可跟随主色
- 主要通过边框和文字颜色表达，不使用大面积主色填充

### 5.9 卡片 Radio 禁用
- 背景 `#F4F6F8` | 边框 25% `#000000` | 文字/图标 25% `#000000` | 鼠标 `not-allowed`

### 5.10 卡片 Radio 选中禁用
- 背景 `#F4F6F8` | 边框 45% `#9B7235` | 文字/图标 45% `#9B7235` | 鼠标 `not-allowed`

### 5.11 后置说明 icon Tooltip
- 说明 icon 尺寸 `16px × 16px` | 文字与 icon 间距 `8px` | 默认 45% `#000000` | Hover `#9B7235` | Tooltip 最大宽度 `280px`
- Tooltip 在 Hover 到说明 icon 时展示，非 Hover 整个选项时展示

### 5.12 辅助说明文字（卡片）
- 字号 `12px` | 行高 `20px` | 颜色 45% `#000000` | 与主文案垂直间距 `2px`
