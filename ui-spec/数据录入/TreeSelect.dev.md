# 树选择器 / TreeSelect - 前端样式规范

> 面向对象：前端开发。本文档仅包含组件的 UI 样式规则和状态样式，用于指导 antd 组件样式调整以达到设计规范要求。

## 1. 组件结构概览

TreeSelect 由**输入区域**和**下拉树面板**两部分组成。

**输入区域**：复用 Select 选择器的输入框规范（背景色、边框色、Placeholder、图标色、多选标签等均与 Select 一致），包含容器外框、Placeholder、已选节点、多选标签、折叠标签（+N）、清除图标、展开图标。

**下拉树面板**包含：树面板容器、展开/收起图标、节点文本、节点缩进、Checkbox（多选模式）、选中状态、半选状态、Hover 状态、禁用节点、Loading 状态、空状态。

## 2. 尺寸与间距规范

### 2.1 输入框（复用 Select 规范）

| 尺寸 | 高度 | 左右内边距 | 文字字号 | 文字行高 | 圆角 | 边框 |
|---|---|---|---|---|---|---|
| 默认 | `32px` | `12px` | `14px` | `22px` | `2px` | `1px` |
| 大尺寸 | `40px` | `12px` | `14px` | `22px` | `2px` | `1px` |

右侧图标视觉尺寸 `16px × 16px`，与输入框右边界间距 `12px`，输入内容与图标区域安全间距 `8px`。

多选标签高度 `24px`，左右内边距 `8px`，标签间距 `4px`，关闭图标 `12px × 12px`。

### 2.2 树面板

| 项目 | 数值 |
|---|---:|
| 面板背景色 | `#FFFFFF` |
| 面板圆角 | `2px` 或 `4px` |
| 面板边框颜色 | 25% `#000000` |
| 面板阴影 | 轻量阴影 |
| 面板最小宽度 | 不小于输入框宽度 |
| 节点高度 | `32px` |
| 节点左右内边距 | `12px` |
| 节点文字字号 | `14px` |
| 节点文字行高 | `22px` |
| 一级节点左侧基础缩进 | `12px` |
| 每级缩进增量 | `24px` |
| 展开/收起图标尺寸 | `16px × 16px` |
| 展开图标与节点文字间距 | `8px` |

### 2.3 多选 Checkbox

| 项目 | 数值 |
|---|---:|
| Checkbox 尺寸 | `16px × 16px` |
| Checkbox 与节点文字间距 | `8px` |

## 5. 状态样式规范

### 5.1 输入框状态
输入框全部状态（默认、已选择、Hover、Focus、禁用、Warning、Error、Loading）复用 Select 选择器输入框样式，参见 [Select.dev.md](Select.dev.md)。

### 5.2 普通节点
- 背景：透明 | 文字：85% `#000000` | 展开图标：45% `#000000` | 鼠标：`pointer`

### 5.3 Hover 节点
- 背景：`#F4F6F8` | 文字：85% `#000000` | 展开图标：45% `#000000` | 鼠标：`pointer`

### 5.4 单选节点选中
- 不展示 Checkbox
- 背景：`#FFFCF0` | 文字：`#9B7235` | 展开图标：`#9B7235` | 鼠标：`pointer`

### 5.5 多选未选中
- Checkbox 背景 `#FFFFFF` | Checkbox 边框 25% `#000000` | 文字 85% `#000000` | 节点背景透明 | 鼠标 `pointer`

### 5.6 多选选中
- Checkbox 背景 `#9B7235` | Checkbox 边框 `#9B7235` | 勾选 `#FFFFFF` | 文字 85% `#000000` | 节点背景 `#FFFCF0` | 鼠标 `pointer`

### 5.7 半选（父节点部分子节点选中）
- Checkbox 背景 `#9B7235` | Checkbox 边框 `#9B7235` | 内部白色横线 | 文字 85% `#000000` | 节点背景透明或 `#FFFCF0` | 鼠标 `pointer`

### 5.8 禁用节点
- 文字 25% `#000000` | Checkbox 边框 25% `#000000` | 展开图标 25% `#000000` | 鼠标 `not-allowed` | 不出现 Hover 反馈

### 5.9 Loading
- 输入框 Loading 复用 Select 规范 | Loading 图标 `#1874FF` | 树面板中可在对应节点展示加载提示
