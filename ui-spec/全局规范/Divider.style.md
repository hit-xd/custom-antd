# Divider — 样式要点（供开发使用）

## 基本样式
- 线高：1px
- 线色：rgba(0,0,0,0.15)
- 水平线：width: 100% (随父容器自适应)
- 垂直线：width: 1px; 高度按内容自适应，建议 12px-16px

## 类型
- 实线：border-top: 1px solid rgba(0,0,0,0.15)
- 虚线：border-top: 1px dashed rgba(0,0,0,0.15)

## 间距（与上下元素）
- 常规：16px
- 紧凑：8px
- 模块间：24px
- 带文字分割：文字与线之间间距 8px，文字字号 14px，字重 Regular，行高 22px，文字颜色 rgba(0,0,0,0.45)

## 使用要点（仅样式）
- 分割线优先保持低视觉权重，避免使用深色或加粗。
- 行内操作的垂直分割线应与文字或按钮垂直居中，左右间距 8px-12px。
