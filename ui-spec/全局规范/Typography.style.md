# Typography — 样式要点（供开发使用）

## 字体族（font-family）
font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", "Helvetica Neue", Helvetica, Arial, sans-serif;

## 字号与行高映射
- 32px / 40px — 重要数字（Bold 500）
- 28px / 36px — 强调数字（Bold 500）
- 24px / 32px — 核心数字（Bold 500）
- 20px / 28px — 重点文字（Bold 500）
- 18px / 26px — 页面标题（Bold 500）
- 16px / 24px — 次级标题（Bold 500）
- 15px / 28px — 资产配置正文（Bold 500）
- 14px / 22px — 正文（Regular）
- 12px / 20px — 辅助文字（Regular）

## 文字颜色（浅色背景）
- 主文字：rgba(0,0,0,0.85)
- 次文字：rgba(0,0,0,0.65)
- 占位：rgba(0,0,0,0.45)
- 禁用：rgba(0,0,0,0.25)
- 链接 / 品牌色：#1874FF
- 成功：#00C078
- 警告：#FFA000
- 错误：#FF4D4F

## 图标尺寸
- 小：12px
- 常规：16px
- 强调：24px

## 使用要点（仅样式）
- 按钮文字建议 14px / 22px；重要按钮使用 Regular 或 Bold 根据场景调整。
- 标题与正文字号数量控制在 3-4 个主要层级内。
- 深色背景下文字使用白色系（100% / 65% / 25%）对应上面透明度层级。
