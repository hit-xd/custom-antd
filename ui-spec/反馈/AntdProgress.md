# Progress 进度条

**组件：** Progress | **分类：** 反馈

---

## 线性进度条

| 属性 | 数值 |
|------|------|
| 高度 | `8px`（默认） |
| 宽度 | 自适应 |
| 圆角 | `4px` |
| 百分比字号 | `12px` Regular |
| 文字与进度条间距 | `8px` |

### 状态色

| 状态 | 填充色 | 百分比文字 |
|------|--------|-----------|
| normal（进行中） | `#1874FF` | `#000000` 45% |
| success（100%） | `#00C078` | `#000000` 85%，右侧显示 `16px` 绿色勾选图标 |
| exception | `#FF4D4F` | `#000000` 45% |
| active | `#1874FF`（带动画） | — |

轨道色：`#1874FF`（底色轨道）

---

## 尺寸

| 尺寸 | 高度 |
|------|------|
| small | 等比缩小 |
| default | `8px` |
| 自定义 | 推荐 ≤ `20px`，圆角随高度等比 |

---

## 圆形 / 仪表盘

| 类型 | 说明 |
|------|------|
| 圆形 | 圆环宽度与线性高度一致，中心显示百分比 |
| 仪表盘 | 半弧形，两端缺口，中心显示百分比 |

---

## 自定义颜色（strokeColor）

| 语义 | 色值 |
|------|------|
| 成功 | `#00C078` |
| 信息 | `#1874FF` |
| 警告 | `#FFA000` |

渐变方向：线性从左至右，圆形顺时针。
