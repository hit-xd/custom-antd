# Spin 加载中

## 基础加载

| 属性 | 数值 |
|------|------|
| 显示方式 | `inline-block` |
| 指示器尺寸 | `32px` |
| 指示器颜色 | `#1874FF` |

| size | 说明 |
|------|------|
| small | 紧凑场景 |
