# Alert 警告提示

**组件：** Alert | **分类：** 反馈

---

## 基本提示

| 属性 | 数值 |
|------|------|
| 宽度 | 自适应，填充容器 |
| 水平内边距 | `12px` |
| 垂直内边距 | `8px` |
| 圆角 | `2px` |
| 标题字号 | `14px` |
| 标题字重 | Regular |
| 标题行高 | `22px` |
| 图标与文字间距 | `8px` |

### 状态色

| 类型 | 图标 | 图标尺寸 | 标题文字 | 背景色 | 边框色 |
|------|------|----------|----------|--------|--------|
| success | 绿色圆形勾选 | `16×16px` | `#000000` 85% | `#E6FFF1` | `#00C078` |
| info | 蓝色圆形信息 | `16×16px` | `#000000` 65% | `#E6F4FF` | `#1874FF` |
| warning | 黄色三角形感叹号 | — | `#000000` 65% | `#FFFAE6` | `#FFA000` |
| error | 红色圆形叉号 | `16×16px` | `#000000` 65% | `#FFF1F0` | `#FF4D4F` |

---

## 带描述的提示

| 属性 | 数值 |
|------|------|
| 标题与描述间距 | `4px` |
| 标题字号 | `16px` |
| 标题字重 | SemiBold |
| 标题行高 | `24px` |
| 标题颜色 | `#000000` 85% |
| 描述字号 | `14px` |
| 描述字重 | Regular |
| 描述行高 | `22px` |
| 描述颜色 | `#000000` 65% |
| 图标对齐 | `align-items: flex-start`（图标与标题顶部齐平） |
| 内边距 | 上 `16px`，左/右/下 `12px` |

---

## 可关闭提示

| 属性 | 数值 |
|------|------|
| 关闭按钮尺寸 | `12×12px` |
| 关闭按钮颜色 | `#000000` 45% |
| 图标尺寸 | `16×16px` |

带描述可关闭：左侧图标、标题与右侧关闭按钮同一水平线。

---

## 顶部公告（banner）

| 属性 | 数值 |
|------|------|
| 模式 | 无边框，宽度占满容器 |
| 默认类型 | info |
| 默认图标 | 信息图标，可关闭 |

---

## 操作按钮

| 属性 | 数值 |
|------|------|
| 位置 | 提示内容右侧，与描述文字水平对齐 |
| 与容器右侧间距 | `12px` |
| 按钮类型 | link 文字链接 |

---

## 设计 Token

### 间距

| Token | 数值 |
|-------|------|
| 水平内边距 | `12px` |
| 垂直内边距 | `8px` |
| 图标与文字间距 | `8px` |
| 内容与右侧间距 | `12px` |
| 标题与描述间距 | `4px` |

### 字体

| 类型 | 字号 | 字重 | 行高 | 颜色 |
|------|------|------|------|------|
| 标题（带描述） | `16px` | Bold | `24px` | 85% #000000 |
| 标题（无描述） | `14px` | Regular | `22px` | 85% #000000 |
| 描述文字 | `14px` | Regular | `22px` | 65% #000000 |
| 按钮文字 | `14px` | Regular | `22px` | `#1874FF` |
