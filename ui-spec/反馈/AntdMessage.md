# Message 全局提示

**组件：** Message | **分类：** 反馈

---

## 整体样式

| 属性 | 数值 |
|------|------|
| 弹出位置 | 页面顶部，距顶 `8px` |
| 宽度 | 自适应 |
| 水平内边距 | `20px` |
| 垂直内边距 | `10px` |
| 圆角 | `4px` |
| 字号 | `14px` Regular |
| 图标与文字 | 垂直居中 |

---

## 状态色

| 类型 | 图标 | 图标色 | 文字色 |
|------|------|--------|--------|
| success | 绿色圆形勾选 | `#00C078` | `#000000` 85% |
| info | 蓝色圆形信息 | `#1874FF` | `#000000` 85% |
| warning | 黄色三角形感叹号 | `#FFA000` | `#000000` 85% |
| error | 红色圆形叉号 | `#FF4D4F` | `#000000` 85% |

图标尺寸：`16×16px`

---

## 加载中

| 属性 | 数值 |
|------|------|
| 图标 | 旋转加载动画 |
| 默认关闭 | `1.5s` |
| 长时操作 | `duration: 0`，手动关闭 |

---

## 设计 Token

### 间距

| Token | 数值 |
|-------|------|
| 距顶部 | `8px` |
| 水平内边距 | `20px` |
| 垂直内边距 | `10px` |
| 圆角 | `4px` |

### 字体

| 类型 | 字号 | 字重 | 颜色 |
|------|------|------|------|
| 提示正文 | `14px` | Regular | 85% #000000 |

### 行为参数

| 参数 | 默认值 |
|------|--------|
| duration | `3s`（加载中 `1.5s`） |
| duration: 0 | 不自动关闭 |
