# Drawer 抽屉

**组件：** Drawer | **分类：** 反馈

---

## 尺寸

| 方向 | 默认尺寸 | 最大限制 |
|------|----------|----------|
| 右侧（默认） | 宽 `378px` | 宽 ≤ 屏幕 75% 或 `1200px` |
| 左侧 | 宽 `378px` | 同右侧 |
| 顶部 | 高 `378px` | 高 ≤ 屏幕 50% |
| 底部 | 高 `378px` | — |

常用宽度：`480px` / `560px` / `720px`

## 响应式

| 断点 | 右侧抽屉宽度 |
|------|-------------|
| ≥ `1440px` | 建议 `480px` 或 `560px` |
| < `768px` | 屏幕宽度 85% |

---

## 滚动条

| 属性 | 数值 |
|------|------|
