# Skeleton 骨架屏

**组件：** Skeleton | **分类：** 反馈

---

## 基础骨架屏

| 属性 | 数值 |
|------|------|
| 显示方式 | `inline-block`，宽度自适应 |
| 标题宽度 | `200px` |
| 段落行高 | `16px` |
| 标题与段落间距 | `16px` |
| 圆角 | `4px` |
| 动画 | `active: true`，从左至右脉冲扫描 |

结构：标题占位条 + 多行段落占位条；可选左侧圆形头像占位（与标题垂直居中）。

---

## 子组件占位

| 组件 | 默认宽度 | 高度 |
|------|----------|------|
| Skeleton.Button | `120px`（可自定义） | 同标准按钮 |
| Skeleton.Input | `240px`（可自定义） | 同标准输入框 |

自定义宽度：`style={{ width }}`

---

## 列表项占位

| 属性 | 数值 |
|------|------|
| 结构 | 左侧圆形头像 + 右侧标题/段落 |
| 列表项间距 | `16px` |
